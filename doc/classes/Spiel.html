neuer_Apfel(wegx=-1,wegy=-1)

Positioniert einen neuen Apfel so, dass auf den acht Feldern um den Apfel nicht ist. Werden Koordinaten uebergeben, wird dort der Apfel aus dem @aepfel Array entfernt.
